# AI API Assistant (LLM Powered)

A simple Spring Boot backend that integrates a Large Language Model (LLM) API.

Users can send a question and receive an AI generated response.

## Features

- LLM API integration
- Prompt engineering
- REST API backend
- Simple AI assistant

## API

POST /ai/ask

Example Request

{
  "question": "Explain microservices"
}

Example Response

Microservices is an architecture where applications are split into small independent services.

## Tech Stack

Spring Boot  
LLM API (Gemini/OpenAI)  
REST APIs
